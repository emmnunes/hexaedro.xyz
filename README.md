# hexaedro.xyz

Website for Pedro Vaz's Hexaedro project, circa 2012.

## Deploying

Install the [Heroku Toolbelt](https://toolbelt.heroku.com/).

```sh
$ git clone git@github.com:emmnunes/hexaedro.xyz.git
$ cd hexaedro.xyz
$ heroku create
$ git push heroku master
$ heroku open
```
